makesweird.art css guide

This page is a super quick intro into customizing your profile, it also provides some templates for you to get started!

So you want to customize your profile? great! You can do so via CSS. Your profile has a custom css field near the bottom. Whatever you type there changes how your profile looks. If you've never written css before, don't worry! It's very simple and fun to play around with.

The pattern is always: what { property: value; }

To change the background and text color of your whole page:

yourname makes weird art.

streak: 3 · 7 logged

body {
  background: #1a1a2e;
  color: #e0d8c8;
}

make your username a different color:

yourname makes weird art.

#uname { color: #ff9500; }

style your bio text:

i love making weird art, the weirder the better!

#bio {
  font-style: italic;
  line-height: 1.8;
  opacity: .7;
}

turn your links into buttons:

instagram portfolio 
#links a {
  display: block;
  background: #ff9500;
  color: #1a1a1a;
  padding: .5rem 1rem;
  text-decoration: none;
  text-align: center;
}
#links a:hover { background: #ffb040; }

color your dots:

your badge (the embeddable iframe) shares your css automatically. to style it differently, use .badge:

.badge #name { color: #ff9500; }
.badge #streak { color: #ff9500; }
.badge #meta a { color: #ff9500; }

embed it on your own site:

<iframe
  src="https://makesweird.art/u/yourname/badge"
  width="380" height="100"
  frameborder="0" scrolling="no"
></iframe>

2048 character limit. dots use currentColor — set color on #dots to change them all at once.

new to css? mdn has a great beginner guide. htmlcolorcodes.com is handy for picking colors.
